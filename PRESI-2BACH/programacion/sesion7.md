## Repaso: Tipos de Consultas SQL
- UPDATE: Actualización de registros existentes
- DELETE: Eliminación de registros

---

## Consulta UPDATE
- Ejemplo: `UPDATE Productos SET Precio = 15 WHERE ID = 1;`

---

## Consulta DELETE
- Ejemplo: `DELETE FROM Productos WHERE ID = 1;`

---
